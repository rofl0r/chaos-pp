The array/to_list.h header defines a macro that converts an array to a list.

Usage

#include <chaos/preprocessor/array/to_list.h>

Contents

  • CHAOS_PP_ARRAY_TO_LIST

CHAOS_PP_ARRAY_TO_LIST

The CHAOS_PP_ARRAY_TO_LIST macro converts an array to a list.

Usage 

CHAOS_PP_ARRAY_TO_LIST(array)
array

The array to be converted.

Remarks

This macro expands to a list containing the elements of array.

The array argument must be an array containing no more than CHAOS_PP_LIMIT_TUPLE elements.

See Also

  • CHAOS_PP_ARRAY_TO_SEQ
  • CHAOS_PP_ARRAY_TO_STRING
  • CHAOS_PP_ARRAY_TO_TUPLE
  • CHAOS_PP_CAST

Sample Code

#include <chaos/preprocessor/array/to_list.h>

#define DATA() (3, (x, y, z))

CHAOS_PP_ARRAY_TO_LIST(DATA()) // (x, (y, (z, ...)))

CHAOS_PP_ARRAY_TO_LIST_ID

The CHAOS_PP_ARRAY_TO_LIST_ID macro indirectly expands to CHAOS_PP_ARRAY_TO_LIST.

Usage 

CHAOS_PP_ARRAY_TO_LIST_ID()

Remarks

This macro effectively allows CHAOS_PP_ARRAY_TO_LIST to pass through itself without becoming disabled.

CHAOS_PP_ARRAY_TO_LIST_

The CHAOS_PP_ARRAY_TO_LIST_ macro expands to a λ expression representing CHAOS_PP_ARRAY_TO_LIST.

Usage - C99 Specific 

CHAOS_PP_ARRAY_TO_LIST_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_ARRAY_TO_LIST_ID)()
