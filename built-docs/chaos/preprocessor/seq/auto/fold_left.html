The seq/auto/fold_left.h header defines an algorithm that accumulates the elements of a sequence from the first element to the last element.

Usage

#include <chaos/preprocessor/seq/auto/fold_left.h>

Contents

  • CHAOS_PP_SEQ_AUTO_FOLD_LEFT

CHAOS_PP_SEQ_AUTO_FOLD_LEFT

The CHAOS_PP_SEQ_AUTO_FOLD_LEFT macro accumulates the elements of its second argument with the operation specified by its first argument.

Usage - C99 Specific 

CHAOS_PP_SEQ_AUTO_FOLD_LEFT(op, seq, ...)
op

The operation that is to be used to accumulate the elements of seq. This operation must be either a λ expression or a macro name that refers to a macro with a signature compatible with the following pseudo-signature: 

op(s, elem†, state‡)

† The elem pseudo-parameter must be the number of parameters that corresponds to the arity of each element in seq.

‡ The state pseudo-parameter must be either ... or the number of parameters that corresponds to the number of arguments in the auxiliary data, __VA_ARGS__.

[seq]

The sequence whose elements are to be accumulated.

[...]

The initial state of the accumulation.

Remarks

The CHAOS_PP_SEQ_AUTO_FOLD_LEFT macro is a higher-order thunk that deduces the current higher-order entry point and invokes CHAOS_PP_SEQ_FOLD_LEFT.

This macro may not be used if bypass semantics are in effect.

Sample Code

#include <chaos/preprocessor/cat.h>
#include <chaos/preprocessor/config.h>
#include <chaos/preprocessor/lambda/ops.h>
#include <chaos/preprocessor/seq/auto/fold_left.h>

#if CHAOS_PP_VARIADICS

??=include CHAOS_PP_PLACEHOLDERS(1)

CHAOS_PP_SEQ_AUTO_FOLD_LEFT(CHAOS_PP_CAT_(_2, _1), (a)(b)(c)(d),) // abcd

??=include CHAOS_PP_PLACEHOLDERS(0)

#endif

CHAOS_PP_SEQ_AUTO_FOLD_LEFT_ID

The CHAOS_PP_SEQ_AUTO_FOLD_LEFT_ID macro indirectly expands to CHAOS_PP_SEQ_AUTO_FOLD_LEFT.

Usage - C99 Specific 

CHAOS_PP_SEQ_AUTO_FOLD_LEFT_ID()

Remarks

This macro effectively allows CHAOS_PP_SEQ_AUTO_FOLD_LEFT to be passed around without expansion.

CHAOS_PP_SEQ_AUTO_FOLD_LEFT_

The CHAOS_PP_SEQ_AUTO_FOLD_LEFT_ macro expands to a λ expression representing CHAOS_PP_SEQ_AUTO_FOLD_LEFT.

Usage - C99 Specific 

CHAOS_PP_SEQ_AUTO_FOLD_LEFT_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_SEQ_AUTO_FOLD_LEFT_ID)()
