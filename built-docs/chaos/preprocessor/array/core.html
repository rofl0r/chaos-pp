CHAOS_PP_ARRAY

The CHAOS_PP_ARRAY macro designates the name of the array data structure.

Usage 

CHAOS_PP_ARRAY

Remarks

This macro expands to CHAOS_PP_ARRAY. In other words, the identifier CHAOS_PP_ARRAY is defined as itself.

CHAOS_PP_ARRAY_ID

The CHAOS_PP_ARRAY_ID macro indirectly expands to CHAOS_PP_ARRAY.

CHAOS_PP_ARRAY_TAG

Usage

The tag used to distinguish between data structures.

Remarks

This macro is used by generic algorithms to compare the types of generic data structures. It should be used for no other purpose.

CHAOS_PP_ARRAY_CONS

The CHAOS_PP_ARRAY_CONS macro appends an element to the head of an array.

Usage - C99 Specific 

CHAOS_PP_ARRAY_CONS(array, x)
array

The array to which x is to be appended.

[x]

The element to be appended to the head of array.

Usage - C/C++ Specific 

CHAOS_PP_ARRAY_CONS(array, x)
array

The array to which x is to be appended.

x

The element to be appended to the head of array.

Remarks

This macro expands to an array containing the elements of array with x appended to the head.

In C and C++, array must be an array containing less than CHAOS_PP_LIMIT_TUPLE elements. In C99, there is no such restriction.

See Also

  • CHAOS_PP_CONS
  • CHAOS_PP_LIST_CONS
  • CHAOS_PP_SEQ_CONS
  • CHAOS_PP_STRING_CONS
  • CHAOS_PP_TUPLE_CONS
  • CHAOS_PP_VARIADIC_SEQ_CONS

Sample Code

#include <chaos/preprocessor/array/core.h>

#define DATA() (2, (b, c))

CHAOS_PP_ARRAY_CONS(DATA(), a) // (3, (a, b, c))

CHAOS_PP_ARRAY_ITEM

The CHAOS_PP_ARRAY_ITEM macro is an identity macro used to encapsulate elements.

Usage - C99 Specific 

CHAOS_PP_ARRAY_ITEM(x)
[x]

The element of an array to be encapsulated.

Usage - C/C++ Specific 

CHAOS_PP_ARRAY_ITEM(x)
x

The element of an array to be encapsulated.

Remarks

This macro is used by generic algorithms to encapsulate elements before they are passed to user-defined macros. Elements in an array cannot be variadic. Therefore, this macro is a simple identity macro.

CHAOS_PP_ARRAY_HEAD

The CHAOS_PP_ARRAY_HEAD macro extracts the first element from an array.

Usage 

CHAOS_PP_ARRAY_HEAD(array)
array

The non-nil array whose first element is to be extracted.

Remarks

This macro expands to the first element contained in array.

In C and C++, array must contain less than CHAOS_PP_LIMIT_TUPLE elements. In C99, there is no such restriction.

CHAOS_PP_ARRAY_TAIL

The CHAOS_PP_ARRAY_TAIL macro removes the first element from an array.

Usage 

CHAOS_PP_ARRAY_TAIL(array)
array

The non-nil array whose first element is to be removed.

Remarks

This macro expands to an array containing all of the elements of array except the first.

In C and C++, array must contain less than CHAOS_PP_LIMIT_TUPLE elements. In C99, there is no such restriction.

CHAOS_PP_ARRAY_IS_CONS

The CHAOS_PP_ARRAY_IS_CONS macro is a predicate that tests whether an array is non-nil.

Usage 

CHAOS_PP_ARRAY_IS_CONS(array)
array

The array to be tested.

Remarks

If array is non-nil, this macro expands to 1. Otherwise, it expands to 0.
