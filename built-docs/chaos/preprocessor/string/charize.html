The string/charize.h header defines a macro that converts a character to a character literal.

Usage

#include <chaos/preprocessor/string/charize.h>

Contents

  • CHAOS_PP_CHARIZE

CHAOS_PP_CHARIZE

The CHAOS_PP_CHARIZE macro converts its argument to a character literal.

Usage 

CHAOS_PP_CHARIZE(c)
c

The character to be converted. Valid values are the underscore (_), the decimal numerals (0 through 9), and the lowercase and uppercase letters (a through z and A through Z).

Remarks

This macro is not a general-purpose "charizing" macro. Input to this macro is restricted to characters that can be elements in a string data structure. For example, 

CHAOS_PP_CHARIZE(+) // invalid

Sample Code

#include <chaos/preprocessor/algorithm/for_each.h>
#include <chaos/preprocessor/recursion/expr.h>
#include <chaos/preprocessor/string/charize.h>
#include <chaos/preprocessor/string/core.h>

#define TO_CHAR_SEQ(string) \
    CHAOS_PP_EXPR(CHAOS_PP_FOR_EACH( \
        TO_CHAR_SEQ_I, (CHAOS_PP_STRING) string, ~ \
    )) \
    /**/

#define TO_CHAR_SEQ_I(s, c, _) (CHAOS_PP_CHARIZE(c))

TO_CHAR_SEQ(t e m p l a t e) // ('t')('e')('m')('p')('l')('a')('t')('e')

CHAOS_PP_CHARIZE_ID

The CHAOS_PP_CHARIZE_ID macro indirectly expands to CHAOS_PP_CHARIZE.

Usage 

CHAOS_PP_CHARIZE_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_CHARIZE_

The CHAOS_PP_CHARIZE_ macro expands to a λ expression representing CHAOS_PP_CHARIZE.

Usage - C99 Specific 

CHAOS_PP_CHARIZE_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_CHARIZE)
