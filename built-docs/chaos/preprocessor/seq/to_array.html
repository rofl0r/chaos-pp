The seq/to_array.h header defines a macro that converts a sequence to an array.

Usage

#include <chaos/preprocessor/seq/to_array.h>

Contents

  • CHAOS_PP_SEQ_TO_ARRAY

CHAOS_PP_SEQ_TO_ARRAY

The CHAOS_PP_SEQ_TO_ARRAY macro converts a sequence to an array.

Usage - C99 Specific 

CHAOS_PP_SEQ_TO_ARRAY(seq)
[seq]

The sequence to be converted.

Usage - C/C++ Specific 

CHAOS_PP_SEQ_TO_ARRAY(seq)
seq

The sequence to be converted.

Remarks

This macro expands to a array containing the elements of seq.

In C99, sequences can contain elements of arities other than one. If such a sequence is converted to an array, the resulting array will be invalid. 

CHAOS_PP_SEQ_TO_ARRAY((a, b)(c)) // invalid

This macro makes no attempt to diagnose that situation.

  • CHAOS_PP_SEQ_TO_LIST
  • CHAOS_PP_SEQ_TO_STRING

Sample Code

#include <chaos/preprocessor/seq/to_array.h>

CHAOS_PP_SEQ_TO_ARRAY((a)(b)(c)(d)) // (4, (a, b, c, d))

CHAOS_PP_SEQ_TO_ARRAY_ID

The CHAOS_PP_SEQ_TO_ARRAY_ID macro indirectly expands to CHAOS_PP_SEQ_TO_ARRAY.

Usage 

CHAOS_PP_SEQ_TO_ARRAY_ID()

Remarks

This macro effectively allows CHAOS_PP_SEQ_TO_ARRAY to pass through itself without becoming disabled.

CHAOS_PP_SEQ_TO_ARRAY_

The CHAOS_PP_SEQ_TO_ARRAY_ macro expands to a λ expression representing CHAOS_PP_SEQ_TO_ARRAY.

Usage - C99 Specific 

CHAOS_PP_SEQ_TO_ARRAY_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_SEQ_TO_ARRAY_ID)()
