The recursion/context.h header defines macros used to store and retrieve context data.

Usage

#include <chaos/preprocessor/recursion/context.h>

Contents

  • CHAOS_PP_ABSOLUTE
  • CHAOS_PP_CONTEXT
  • CHAOS_PP_RELATIVE
  • CHAOS_PP_RETRIEVE

CHAOS_PP_CONTEXT

The CHAOS_PP_CONTEXT creates a retrievable context.

Usage - C99 Specific 

CHAOS_PP_CONTEXT(bit)(...)
bit

The context data to be created. Valid values are 0 and 1.

[...]

The deferred expression to be expanded within the context.

Usage - C/C++ Specific 

CHAOS_PP_CONTEXT(bit)(x)
bit

The context data to be created. Valid values are 0 and 1.

x

The deferred expression to be expanded within the context.

Remarks

This macro introduces a context value which can be retrieved via CHAOS_PP_RETRIEVE, CHAOS_PP_ABSOLUTE, or CHAOS_PP_RELATIVE. If bit expands to 0, then a directly nested invocation of CHAOS_PP_RETRIEVE inside the x or __VA_ARGS__ will expand to 0. Otherwise, it will expand to 1. 

#define MACRO() CHAOS_PP_RETRIEVE()

CHAOS_PP_CONTEXT(0)(CHAOS_PP_DEFER(MACRO)()) // 0
CHAOS_PP_CONTEXT(1)(CHAOS_PP_DEFER(MACRO)()) // 1

This macro employs automatic recursion, and allows the establishment of up to CHAOS_PP_LIMIT_CONTEXT nested contexts. The most current context can be accessed with CHAOS_PP_RETRIEVE, contexts relative to the current context can be accessed with CHAOS_PP_RELATIVE, and absolute contexts can be accessed with CHAOS_PP_ABSOLUTE.

Sample Code

#include <chaos/preprocessor/recursion/basic.h>
#include <chaos/preprocessor/recursion/context.h>

#define A() CHAOS_PP_CONTEXT(0)(CHAOS_PP_DEFER(B)())
#define B() CHAOS_PP_CONTEXT(1)(CHAOS_PP_DEFER(C)())
#define C() \
    CHAOS_PP_ABSOLUTE(1) \
        CHAOS_PP_ABSOLUTE(2) \
            CHAOS_PP_RETRIEVE() \
    /**/

CHAOS_PP_CONTEXT(1)(CHAOS_PP_DEFER(A)()) // 1 0 1

CHAOS_PP_ABSOLUTE

The CHAOS_PP_ABSOLUTE macro retrieves the context value of a context at an absolute position.

Usage 

CHAOS_PP_ABSOLUTE(n)
n

The index of the context frame whose context value is to be retrieved. Valid values range from 1 to CHAOS_PP_LIMIT_CONTEXT.

Remarks

This macro expands to the context value at position n. If no context is present at position n, this macro expands to nothing.

Sample Code

#include <chaos/preprocessor/recursion/basic.h>
#include <chaos/preprocessor/recursion/context.h>

#define A() CHAOS_PP_CONTEXT(0)(CHAOS_PP_DEFER(B)())
#define B() CHAOS_PP_CONTEXT(1)(CHAOS_PP_DEFER(C)())
#define C() \
    CHAOS_PP_ABSOLUTE(1) \
        CHAOS_PP_ABSOLUTE(2) \
            CHAOS_PP_ABSOLUTE(3) \
    /**/

CHAOS_PP_CONTEXT(0)(CHAOS_PP_DEFER(A)()) // 0 0 1

CHAOS_PP_RELATIVE

The CHAOS_PP_RELATIVE macro retrieves the context value of a context relative to the current context.

Usage 

CHAOS_PP_RELATIVE(n)
n

The offset from the current context to the context frame whose context value is to be retrieved. Valid values range 0 to CHAOS_PP_LIMIT_CONTEXT - 1.

Remarks

This macro expands to the context value of the context n outer frames from the current context frame. In other words, CHAOS_PP_RELATIVE(0) expands to the current context value, CHAOS_PP_RELATIVE(1) expands to the next outer context, CHAOS_PP_RELATIVE(2) expands to the next outer context after that, and so on. If no context exists n frames outward, this macro expands to 0.

Sample Code

#include <chaos/preprocessor/recursion/basic.h>
#include <chaos/preprocessor/recursion/context.h>

#define A() CHAOS_PP_CONTEXT(1)(CHAOS_PP_DEFER(B)())
#define B() CHAOS_PP_CONTEXT(0)(CHAOS_PP_DEFER(C)())
#define C() \
    CHAOS_PP_RELATIVE(0) \
        CHAOS_PP_RELATIVE(1) \
            CHAOS_PP_RELATIVE(2) \
    /**/

CHAOS_PP_CONTEXT(1)(CHAOS_PP_DEFER(A)()) // 0 1 1

CHAOS_PP_RETRIEVE

The CHAOS_PP_RETRIEVE macro retrieves the context value of the most current context.

Usage 

CHAOS_PP_RETRIEVE()

Remarks

If the most current context value is 1, this macro expands to 1. Otherwise, it expands to 0.

Sample Code

#include <chaos/preprocessor/control/inline_when.h>
#include <chaos/preprocessor/recursion/basic.h>
#include <chaos/preprocessor/recursion/context.h>

#define A(x) \
    CHAOS_PP_INLINE_WHEN(CHAOS_PP_RETRIEVE())(x B) \
    /**/
#define B(x) \
    CHAOS_PP_INLINE_WHEN(CHAOS_PP_RETRIEVE())(x A) \
    /**/

CHAOS_PP_CONTEXT(1)(
    CHAOS_PP_DEFER(A)(+)(-)(*)(/)(%)
)(...)
// + - * / %
