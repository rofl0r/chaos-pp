The string/to_lower.h header defines an algorithm that converts a string to lowercase.

Usage

#include <chaos/preprocessor/string/to_lower.h>

Contents

CHAOS_PP_TO_LOWER

The CHAOS_PP_TO_LOWER macro converts a character to lowercase.

Usage 

CHAOS_PP_TO_LOWER(c)
c

The character to be converted to lowercase. Valid values are the numerals (0 through 9), the alphabetic characters in the basic source character set (a through z and A through Z), and the underscore (_).

Remarks

If c is an uppercase letter, this macro expands to the letter's lowercase equivalent. Otherwise, it expands to c.

See Also

  • CHAOS_PP_TO_UPPER

Sample Code

#include <chaos/preprocessor/string/to_lower.h>

CHAOS_PP_TO_LOWER(a) // a
CHAOS_PP_TO_LOWER(A) // a
CHAOS_PP_TO_LOWER(0) // 0
CHAOS_PP_TO_LOWER(_) // _

CHAOS_PP_TO_LOWER_ID

The CHAOS_PP_TO_LOWER_ID macro indirectly expands to CHAOS_PP_TO_LOWER.

Usage 

CHAOS_PP_TO_LOWER_ID()

Remarks

This macro effectively allows CHAOS_PP_TO_LOWER to pass through itself without becoming disabled.

CHAOS_PP_TO_LOWER_

The CHAOS_PP_TO_LOWER_ macro expands to a λ expression representing CHAOS_PP_TO_LOWER.

Usage - C99 Specific 

CHAOS_PP_TO_LOWER_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_TO_LOWER_ID)()

CHAOS_PP_STRING_TO_LOWER

The CHAOS_PP_STRING_TO_LOWER macro converts its argument to lowercase.

Usage - C99 Specific 

CHAOS_PP_STRING_TO_LOWER(string)
[string]

The string to be converted to lowercase.

Usage - C/C++ Specific 

CHAOS_PP_STRING_TO_LOWER(string)
string

The string to be converted to lowercase.

Remarks

This macro expands to a string containing the elements of string converted to lowercase. For example, 

CHAOS_PP_STRING_TO_LOWER(C H A O S) // c h a o s

This macro uses bypass semantics. If bypass semantics are already in effect, CHAOS_PP_STRING_TO_LOWER_BYPASS must be used instead.

See Also

  • CHAOS_PP_STRING_TO_UPPER

Sample Code

#include <chaos/preprocessor/algorithm/concat.h>
#include <chaos/preprocessor/string/core.h>
#include <chaos/preprocessor/string/to_lower.h>
#include <chaos/preprocessor/string/to_upper.h>

CHAOS_PP_CONCAT(
    (CHAOS_PP_STRING) CHAOS_PP_STRING_TO_LOWER(
        A B c _ 1 0
    )
)
// abc_10

#define PROPER(string) \
    PROPER_I(CHAOS_PP_STRING_TO_LOWER(string)) \
    /**/
#define PROPER_I(string) \
    CHAOS_PP_TO_UPPER(CHAOS_PP_STRING_HEAD(string)) \
    CHAOS_PP_STRING_TAIL(string) \
    /**/

PROPER(C H A O S) // C h a o s

CHAOS_PP_STRING_TO_LOWER_BYPASS

The CHAOS_PP_STRING_TO_LOWER_BYPASS macro behaves identically to CHAOS_PP_STRING_TO_LOWER except that it is parametized by the bypass recursion state.

Usage - C99 Specific 

CHAOS_PP_STRING_TO_LOWER_BYPASS(s, string)
s

The current bypass recursion state.

[string]

The string to be converted to lowercase.

Usage - C/C++ Specific 

CHAOS_PP_STRING_TO_LOWER_BYPASS(s, string)
s

The current bypass recursion state.

string

The string to be converted to lowercase.

Remarks

If bypass semantics are already in effect, this macro must be used instead of CHAOS_PP_STRING_TO_LOWER.
