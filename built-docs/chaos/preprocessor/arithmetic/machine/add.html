CHAOS_PP_ADD_MACHINE

The CHAOS_PP_ADD_MACHINE macro computes the sum of two values.

Usage - C99 Specific 

CHAOS_PP_ADD_MACHINE(x, y)
x

The first addend. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

y

The second addend. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

Remarks

This macro expands to the sum of x and y. If the result is greater than CHAOS_PP_LIMIT_MAG, it is saturated to CHAOS_PP_LIMIT_MAG.

This macro behaves identically to CHAOS_PP_ADD except that it is implemented with the continuation machine, which can yield significant performance improvements.

This macro may not be used inside a continuation machine instruction. However, the instruction upon which this macro is built, (0xchaos)(0xadd), may be used inside a continuation. 

CHAOS_PP_MACHINE(, 12, (0xchaos)(0xadd), 34, (0xchaos)(0xstop),) // 46

Sample Code

#include <chaos/preprocessor/arithmetic/machine/add.h>
#include <chaos/preprocessor/config.h>

#if CHAOS_PP_VARIADICS

CHAOS_PP_ADD_MACHINE(11, 12) // 23

#endif

CHAOS_PP_ADD_MACHINE_ID

The CHAOS_PP_ADD_MACHINE_ID macro indirectly expands to CHAOS_PP_ADD_MACHINE.

Usage - C99 Specific 

CHAOS_PP_ADD_MACHINE_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_ADD_MACHINE_

The CHAOS_PP_ADD_MACHINE_ macro expands to a λ expression representing CHAOS_PP_ADD_MACHINE.

Usage - C99 Specific 

CHAOS_PP_ADD_MACHINE_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_ADD_MACHINE)
