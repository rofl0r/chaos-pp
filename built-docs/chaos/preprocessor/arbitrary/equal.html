CHAOS_PP_ARBITRARY_EQUAL

The CHAOS_PP_ARBITRARY_EQUAL macro compares two arbitrary-precision values for equality.

Usage 

CHAOS_PP_ARBITRARY_EQUAL(x, y)
x

The first operand.

y

The second operand.

Remarks

If x is equal to y, this macro expands to 1. Otherwise, it expands to 0.

See Also

  • CHAOS_PP_EQUAL

Sample Code

#include <chaos/preprocessor/arbitrary/equal.h>

CHAOS_PP_ARBITRARY_EQUAL((1)(2), (3)(4)) // 0
CHAOS_PP_ARBITRARY_EQUAL((5)(6), (5)(6)) // 1

CHAOS_PP_ARBITRARY_EQUAL_ID

The CHAOS_PP_ARBITRARY_EQUAL_ID macro indirectly expands to CHAOS_PP_ARBITRARY_EQUAL.

Usage 

CHAOS_PP_ARBITRARY_EQUAL_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_ARBITRARY_EQUAL_

The CHAOS_PP_ARBITRARY_EQUAL_ macro expands to a λ expression representing CHAOS_PP_ARBITRARY_EQUAL.

Usage - C99 Specific 

CHAOS_PP_ARBITRARY_EQUAL_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_ARBITRARY_EQUAL)
