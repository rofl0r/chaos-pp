The arbitrary/mul.h header defines an arbitrary-precision multiplication operator.

Usage

#include <chaos/preprocessor/arbitrary/mul.h>

Contents

  • CHAOS_PP_ARBITRARY_MUL

CHAOS_PP_ARBITRARY_MUL

The CHAOS_PP_ARBITRARY_MUL macro computes the product of two arbitrary-precision values.

Usage 

CHAOS_PP_ARBITRARY_MUL(x, y)
x

The multiplicand.

y

The multiplier.

Remarks

This macro expands to an arbitrary-precision value that represents the product of x and y.

See Also

  • CHAOS_PP_MUL
  • CHAOS_PP_MUL_MACHINE

Sample Code

#include <chaos/preprocessor/arbitrary/demote.h>
#include <chaos/preprocessor/arbitrary/mul.h>

CHAOS_PP_ARBITRARY_DEMOTE(CHAOS_PP_ARBITRARY_MUL(
    (9)(8)(7)(6)(5)(4)(3)(2)(1)(0)(9)(8)(7)(6)(5)(4)(3)(2)(1)(0),
    (8)(7)(6)(5)(4)(3)(2)(1)(0)(9)(8)(7)(6)(5)(4)(3)(2)(1)(0)(9)
))
// 8657216899617436368663770766778859929890

CHAOS_PP_ARBITRARY_MUL_ID

The CHAOS_PP_ARBITRARY_MUL_ID macro indirectly expands to CHAOS_PP_ARBITRARY_MUL.

Usage 

CHAOS_PP_ARBITRARY_MUL_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_ARBITRARY_MUL_

The CHAOS_PP_ARBITRARY_MUL_ macro expands to a λ expression representing CHAOS_PP_ARBITRARY_MUL.

Usage - C99 Specific 

CHAOS_PP_ARBITRARY_MUL_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_ARBITRARY_MUL)
