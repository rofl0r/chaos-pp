The detection/is_flag.h header defines a macro that whether a configuration flag is set.

Usage

#include <chaos/preprocessor/detection/is_flag.h>

Contents

  • CHAOS_PP_IS_FLAG

CHAOS_PP_IS_FLAG

The CHAOS_PP_IS_FLAG macro detects whether its argument is defined as 1 or with an empty replacement list.

Usage 

CHAOS_PP_IS_FLAG(id)
id

The identifier to be tested.

Remarks

This macro detects whether its argument is defined as 1 or with an empty replacement list.

The id argument must expand to 0, expand to 1, expand to nothing, or not expand at all. Any other input is undefined.

See Also

  • CHAOS_PP_IS_EMPTY
  • CHAOS_PP_IS_ONE

Sample Code

#include <chaos/preprocessor/detection/is_flag.h>

#define A
#define B 0
#define C 1

CHAOS_PP_IS_FLAG(A) // 1
CHAOS_PP_IS_FLAG(B) // 0
CHAOS_PP_IS_FLAG(C) // 1
CHAOS_PP_IS_FLAG(D) // 0

CHAOS_PP_IS_FLAG_ID

The CHAOS_PP_IS_FLAG_ID macro indirectly expands to CHAOS_PP_IS_FLAG.

Usage 

CHAOS_PP_IS_FLAG_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_IS_FLAG_

The CHAOS_PP_IS_FLAG_ macro expands to a λ expression representing CHAOS_PP_IS_FLAG.

Usage - C99 Specific 

CHAOS_PP_IS_FLAG_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_IS_FLAG)
