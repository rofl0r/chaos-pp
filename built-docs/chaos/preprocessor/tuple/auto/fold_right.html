The tuple/auto/fold_right.h header defines an algorithm that accumulates the elements of a tuple from the last element to the first element.

Usage

#include <chaos/preprocessor/tuple/auto/fold_right.h>

Contents

CHAOS_PP_TUPLE_AUTO_FOLD_RIGHT

The CHAOS_PP_TUPLE_AUTO_FOLD_RIGHT macro accumulates the elements of its second argument with the operation specified by its first argument.

Usage - C99 Specific 

CHAOS_PP_TUPLE_AUTO_FOLD_RIGHT(op, tuple, ...)
op

The operation that is to be used to accumulate the elements of tuple. This operation must be either a λ expression or a macro name that refers to a macro with a signature compatible with the following pseudo-signature: 

op(s, elem, state†)

† The state pseudo-parameter must be either ... or the number of parameters that corresponds to the number of arguments in the accumulation state, __VA_ARGS__.

[tuple]

The tuple whose elements are to be accumulated.

[...]

The initial state of the accumulation.

Remarks

The CHAOS_PP_TUPLE_AUTO_FOLD_RIGHT macro is a higher-order thunk that deduces the current higher-order entry point and invokes CHAOS_PP_TUPLE_FOLD_RIGHT.

This macro may not be used if bypass semantics are in effect.

CHAOS_PP_TUPLE_AUTO_FOLD_RIGHT_ID

The CHAOS_PP_TUPLE_AUTO_FOLD_RIGHT_ID macro indirectly expands to CHAOS_PP_TUPLE_AUTO_FOLD_RIGHT.

Usage - C99 Specific 

CHAOS_PP_TUPLE_AUTO_FOLD_RIGHT_ID()

Remarks

This macro effectively allows CHAOS_PP_TUPLE_AUTO_FOLD_RIGHT to be passed around without expansion.

CHAOS_PP_TUPLE_AUTO_FOLD_RIGHT_

The CHAOS_PP_TUPLE_AUTO_FOLD_RIGHT_ macro expands to a λ expression representing CHAOS_PP_TUPLE_AUTO_FOLD_RIGHT.

Usage - C99 Specific 

CHAOS_PP_TUPLE_AUTO_FOLD_RIGHT_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_TUPLE_AUTO_FOLD_RIGHT_ID)()
