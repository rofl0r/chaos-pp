The extended/variadic_cat.h header defines a macro capable of concatenating a variable number of arguments.

Usage

#include <chaos/preprocessor/extended/variadic_cat.h>

Contents

  • CHAOS_PP_VARIADIC_CAT

CHAOS_PP_VARIADIC_CAT

The CHAOS_PP_VARIADIC_CAT macro concatenates its arguments.

Usage - C99 Specific 

CHAOS_PP_VARIADIC_CAT(...)
[...]

The argument(s) to be concatenated.

Remarks

Each argument in __VA_ARGS__ is allowed to expand prior to concatenation.

This macro can concatenate up to a maximum of CHAOS_PP_LIMIT_VARIADIC_CAT arguments.

Sample Code

#include <chaos/preprocessor/config.h>
#include <chaos/preprocessor/extended/variadic_cat.h>

#if CHAOS_PP_VARIADICS

CHAOS_PP_VARIADIC_CAT(1, 2, 3, 4) // 1234

#endif

CHAOS_PP_VARIADIC_CAT_ID

The CHAOS_PP_VARIADIC_CAT_ID macro indirectly expands to CHAOS_PP_VARIADIC_CAT.

Usage - C99 Specific 

CHAOS_PP_VARIADIC_CAT_ID()

Remarks

This macro effectively allows CHAOS_PP_VARIADIC_CAT to pass through itself without becoming disabled.

CHAOS_PP_VARIADIC_CAT_

The CHAOS_PP_VARIADIC_CAT_ macro expands to a λ expression representing CHAOS_PP_VARIADIC_CAT.

Usage - C99 Specific 

CHAOS_PP_VARIADIC_CAT_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_VARIADIC_CAT_ID)()
