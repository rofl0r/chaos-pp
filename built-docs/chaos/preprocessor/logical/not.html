The logical/not.h header defines a logical NOT operator.

Usage

#include <chaos/preprocessor/logical/not.h>

Contents

  • CHAOS_PP_NOT

CHAOS_PP_NOT

The CHAOS_PP_NOT macro expands to the logical NOT of its operand.

Usage 

CHAOS_PP_NOT(x)
x

The operand whose logical NOT is to be produced. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

Remarks

If x is 0, this macro expands to 1. Otherwise, it expands to 0.

This macro performs a Boolean conversion on its operand. If this conversion is undesirable, CHAOS_PP_COMPL may be used instead.

See Also

  • CHAOS_PP_COMPL

Sample Code

#include <chaos/preprocessor/logical/not.h>

CHAOS_PP_NOT(9) // 0
CHAOS_PP_NOT(0) // 1

CHAOS_PP_NOT(CHAOS_PP_NOT(100)) // 1

CHAOS_PP_NOT_ID

The CHAOS_PP_NOT_ID macro indirectly expands to CHAOS_PP_NOT.

Usage 

CHAOS_PP_NOT_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_NOT_

The CHAOS_PP_NOT_ macro expands to a λ expression representing CHAOS_PP_NOT.

Usage - C99 Specific 

CHAOS_PP_NOT_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_NOT)
