The seq/auto/fold_left_aux.h header defines an algorithm that accumulates the elements of a sequence from the first element to the last element.

Usage

#include <chaos/preprocessor/seq/auto/fold_left_aux.h>

Contents

  • CHAOS_PP_SEQ_AUTO_FOLD_LEFT_AUX

CHAOS_PP_SEQ_AUTO_FOLD_LEFT_AUX

The CHAOS_PP_SEQ_AUTO_FOLD_LEFT_AUX macro accumulates the elements of its second argument with the operation specified by its first argument.

Usage - C99 Specific 

CHAOS_PP_SEQ_AUTO_FOLD_LEFT_AUX(op, seq, data, ...)
op

The operation that is to be used to accumulate the elements of seq. This operation must be either a λ expression or a macro name that refers to a macro with a signature compatible with the following pseudo-signature: 

op(s, elem1, data2, state3)

1 The elem pseudo-parameter must be the number of parameters that corresponds to the arity of each element in seq.

2 If the auxiliary data is parenthetic, the data pseudo-parameter must be the number of parameters that corresponds to the number of sub-elements in the auxiliary data. Otherwise, it must be a single parameter.

3 The state pseudo-parameter must be either ... or the number of parameters that corresponds to the number of arguments in the auxiliary data, __VA_ARGS__.

[seq]

The sequence whose elements are to be accumulated.

[data]

Auxiliary data to be passed through the algorithm to the operation. This argument may be either a parenthetic expression of any arity or a non-parenthetic expression.

[...]

The initial state of the accumulation.

Remarks

The CHAOS_PP_SEQ_AUTO_FOLD_LEFT_AUX macro is a higher-order thunk that deduces the current higher-order entry point and invokes CHAOS_PP_SEQ_FOLD_LEFT_AUX.

This macro may not be used if bypass semantics are in effect.

CHAOS_PP_SEQ_AUTO_FOLD_LEFT_AUX_ID

The CHAOS_PP_SEQ_AUTO_FOLD_LEFT_AUX_ID macro indirectly expands to CHAOS_PP_SEQ_AUTO_FOLD_LEFT_AUX.

Usage - C99 Specific 

CHAOS_PP_SEQ_AUTO_FOLD_LEFT_AUX_ID()

Remarks

This macro effectively allows CHAOS_PP_SEQ_AUTO_FOLD_LEFT_AUX to be passed around without expansion.

CHAOS_PP_SEQ_AUTO_FOLD_LEFT_AUX_

The CHAOS_PP_SEQ_AUTO_FOLD_LEFT_AUX_ macro expands to a λ expression representing CHAOS_PP_SEQ_AUTO_FOLD_LEFT_AUX.

Usage - C99 Specific 

CHAOS_PP_SEQ_AUTO_FOLD_LEFT_AUX_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_SEQ_AUTO_FOLD_LEFT_AUX_ID)()
