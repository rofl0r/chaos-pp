The comparison/min.h header defines a macro that expands to the minimum of two values.

Usage

#include <chaos/preprocessor/comparison/min.h>

Contents

  • CHAOS_PP_MIN

CHAOS_PP_MIN

The CHAOS_PP_MIN macro expands to the minimum of its two arguments.

Usage 

CHAOS_PP_MIN(x, y)
x

The first operand. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

y

The second operand. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

Remarks

If x is less than or equal to y, this macro expands to x. Otherwise, it expands to y.

See Also

  • CHAOS_PP_MAX

Sample Code

#include <chaos/preprocessor/comparison/min.h>

CHAOS_PP_MIN(2, 4) // 2

CHAOS_PP_MIN_ID

The CHAOS_PP_MIN_ID macro indirectly expands to CHAOS_PP_MIN.

Usage 

CHAOS_PP_MIN_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_MIN_

The CHAOS_PP_MIN_ macro expands to a λ expression representing CHAOS_PP_MIN.

Usage - C99 Specific 

CHAOS_PP_MIN_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_MIN)
