The comparison/less_equal.h header defines a "less-than-or-equal-to" operator.

Usage

#include <chaos/preprocessor/comparison/less_equal.h>

Contents

  • CHAOS_PP_LESS_EQUAL

CHAOS_PP_LESS_EQUAL

The CHAOS_PP_LESS_EQUAL macro tests whether its first operand is less than or equal to its second operand.

Usage 

CHAOS_PP_LESS_EQUAL(x, y)
x

The first operand. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

y

The second operand. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

Remarks

If x is less than or equal to y, this macro expands to 1. Otherwise, it expands to 0.

See Also

  • CHAOS_PP_ARBITRARY_LESS_EQUAL

Sample Code

#include <chaos/preprocessor/comparison/less_equal.h>

CHAOS_PP_LESS_EQUAL(123, 45) // 0
CHAOS_PP_LESS_EQUAL(12, 345) // 1

CHAOS_PP_LESS_EQUAL(10, 10) // 1

CHAOS_PP_LESS_EQUAL_ID

The CHAOS_PP_LESS_EQUAL_ID macro indirectly expands to CHAOS_PP_LESS_EQUAL.

Usage 

CHAOS_PP_LESS_EQUAL_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_LESS_EQUAL_

The CHAOS_PP_LESS_EQUAL_ macro expands to a λ expression representing CHAOS_PP_LESS_EQUAL.

Usage - C99 Specific 

CHAOS_PP_LESS_EQUAL_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_LESS_EQUAL)
