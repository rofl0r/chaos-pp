CHAOS_PP_DIV_MACHINE

The CHAOS_PP_DIV_MACHINE macro computes the quotient produced by dividing two values.

Usage - C99 Specific 

CHAOS_PP_DIV_MACHINE(x, y)
x

The dividend. Valid values range from 0 to CHAOS_PP_LIMIT_MAG.

y

The divisor. Valid values range from 1 to CHAOS_PP_LIMIT_MAG.

Remarks

This macro expands to the quotient produced by dividing x by y. If y is 0, this macro intentionally produces an error.

This macro behaves identically to CHAOS_PP_DIV except that it is implemented with the continuation machine, which can yield significant performance improvements.

This macro may not be used inside a continuation machine instruction. However, the instruction upon which this macro is built, (0xchaos)(0xdiv), may be used inside a continuation. 

CHAOS_PP_MACHINE(, 22, (0xchaos)(0xdiv), 7, (0xchaos)(0xstop),) // 3

CHAOS_PP_DIV_MACHINE_ID

The CHAOS_PP_DIV_MACHINE_ID macro indirectly expands to CHAOS_PP_DIV_MACHINE.

Usage - C99 Specific 

CHAOS_PP_DIV_MACHINE_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_DIV_MACHINE_

The CHAOS_PP_DIV_MACHINE_ macro expands to a λ expression representing CHAOS_PP_DIV_MACHINE.

Usage - C99 Specific 

CHAOS_PP_DIV_MACHINE_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_DIV_MACHINE)
