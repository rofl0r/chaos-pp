The arbitrary/inc.h header defines an arbitrary-precision incrementation operator.

Usage

#include <chaos/preprocessor/arbitrary/inc.h>

Contents

  • CHAOS_PP_ARBITRARY_INC

CHAOS_PP_ARBITRARY_INC

The CHAOS_PP_ARBITRARY_INC macro increments an arbitrary-precision value.

Usage 

CHAOS_PP_ARBITRARY_INC(x)
x

The arbitrary-precision value to be incremented.

Remarks

This macro results in an arbitrary-precision value one greater than x.

See Also

  • CHAOS_PP_ARBITRARY_DEC
  • CHAOS_PP_DEC

Sample Code

#include <chaos/preprocessor/arbitrary/demote.h>
#include <chaos/preprocessor/arbitrary/inc.h>
#include <chaos/preprocessor/arbitrary/literal.h>

CHAOS_PP_ARBITRARY_DEMOTE(
    CHAOS_PP_ARBITRARY_INC(CHAOS_PP_ARBITRARY_LITERAL(1 2 3))
)
// 124

CHAOS_PP_ARBITRARY_DEMOTE(
    CHAOS_PP_ARBITRARY_INC(CHAOS_PP_ARBITRARY_LITERAL((-)1 2 3))
)
// -122

CHAOS_PP_ARBITRARY_DEMOTE(
    CHAOS_PP_ARBITRARY_INC(CHAOS_PP_ARBITRARY_LITERAL((-)1))
)
// 0

CHAOS_PP_ARBITRARY_INC_ID

The CHAOS_PP_ARBITRARY_INC_ID macro indirectly expands to CHAOS_PP_ARBITRARY_INC.

Usage 

CHAOS_PP_ARBITRARY_INC_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_ARBITRARY_INC_

The CHAOS_PP_ARBITRARY_INC_ macro expands to a λ expression representing CHAOS_PP_ARBITRARY_INC.

Usage - C99 Specific 

CHAOS_PP_ARBITRARY_INC_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_ARBITRARY_INC)
