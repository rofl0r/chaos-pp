CHAOS_PP_ARBITRARY_DIV

The CHAOS_PP_ARBITRARY_DIV macro computes the quotient produced by dividing two arbitrary-precision values.

Usage 

CHAOS_PP_ARBITRARY_DIV(x, y)
x

The dividend.

y

The divisor.

Remarks

This macro expands to an arbitrary-precision value that represents the quotient produced by dividing x by y. If y is an arbitrary-precision value representing zero, this macro intentionally produces an error.

Sample Code

#include <chaos/preprocessor/arbitrary/demote.h>
#include <chaos/preprocessor/arbitrary/div.h>

CHAOS_PP_ARBITRARY_DEMOTE(CHAOS_PP_ARBITRARY_DIV(
    (1)(2)(3)(4),
    (5)(0)
))
// 24

CHAOS_PP_ARBITRARY_DIV_ID

The CHAOS_PP_ARBITRARY_DIV_ID macro indirectly expands to CHAOS_PP_ARBITRARY_DIV.

Usage 

CHAOS_PP_ARBITRARY_DIV_ID()

Remarks

This macro exists only for uniformity with other macros that require indirect invocation.

CHAOS_PP_ARBITRARY_DIV_

The CHAOS_PP_ARBITRARY_DIV_ macro expands to a λ expression representing CHAOS_PP_ARBITRARY_DIV.

Usage - C99 Specific 

CHAOS_PP_ARBITRARY_DIV_

Remarks

This macro is a shorthand form of the following λ binding: 

CHAOS_PP_LAMBDA(CHAOS_PP_ARBITRARY_DIV)
